# [NF-SROS] Operating System

A custom-built, functional computer operating system developed completely from scratch. This project demonstrates the implementation of a custom bootloader, kernel logic, and fundamental system processes, created entirely by a 9-year-old developer.

This repository serves as the official open-source codebase for a pending Guinness World Records application for the title of **"Youngest computer operating system (OS) developer"**.

## 🚀 Features
* **Custom Architecture:** Built from the ground up without using existing Linux distributions or pre-made kernels.
* **Bootloader & Kernel:** Handles system initialization and core logic directly upon booting.
* **User Interface:** Features a minimalist CLI (Command Line Interface) / GUI (Graphical User Interface) allowing user interactions.
* **Core Commands:** Supports internal built-in systems such as `Commands: notepad, help, cls. See more in command help`.

## 🛠️ Tech Stack & Tools
* **Development Language:** `C`
* **Emulation / Testing:** Optimized to run smoothly on `QEMU`

## 📦 How to Run / Emulate
To test and run this operating system locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/florianpeter874-wq/NF-SROS.git
   ```
2. `Run the bin file with qemu.`
